Data Safety Declaration

Data Collected:
- Personal Info: Name, Email Address, Phone Number (for account creation and club membership).
- Financial Info: Payment history (processed via Stripe, card details are not stored directly on our servers).
- App Info and Performance: Crash logs, diagnostics (via Firebase Crashlytics).

Data Sharing:
- Analytics data shared with Google Analytics.
- No personal data is sold to third parties.

Security Practices:
- Data is encrypted in transit using HTTPS/TLS.
- Users can request data deletion at any time via the in-app settings or support email.
